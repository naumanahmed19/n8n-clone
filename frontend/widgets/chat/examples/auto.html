N8n Chat Widget - Auto Initialization

Auto initialization example:

The chat widget will automatically initialize when the page loads by detecting the data attributes below.

This example demonstrates automatic initialization of the N8n Chat Widget using data attributes.

<div data-n8n-chat="auto-chat-123" data-api-url="http://localhost:4000" data-theme="light" data-position="bottom-right"></div>

The widget will appear in the bottom-right corner of the page automatically.
