Manual Initialization Loading

This demo shows how to manually initialize the chat widget with full control over its behavior.

<!-- Load widget script --> <script src="http://localhost:3000/widgets/chat/n8n-chat-widget.umd.js"></script> <!-- Initialize with JavaScript --> <script> const widget = new window.N8nChatWidget(); widget.init({ chatId: 'ca37729d-50bd-4d4d-bb3b-fa8d61e3bdd5', apiUrl: 'http://localhost:4000/api', theme: 'light', position: 'bottom-right', onMessage: (message) => { console.log('User sent:', message); }, onResponse: (response) => { console.log('AI responded:', response); }, onError: (error) => { console.error('Chat error:', error); } }); </script>
